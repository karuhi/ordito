クイックスタート

Ordito API を使い始めるための最短手順です。まず 認証 でAPIキーを取得し、Authorization ヘッダーに付与します。公開情報は ステータスページ で確認できます。

INFOこのガイドは 5分 で完了します。Node.js 18 以降を推奨します。

インストール

install.shbash
npm install -g @ordito/cli
ordito --version

最初のリクエスト

  1. キーを設定環境変数 ORDITO_API_KEY にキーを設定します。
  2. リクエスト送信/v1/me を呼び、認証が通ることを確認します。
  3. レート制限を確認レスポンスヘッダーで残り回数を確認します。詳細は レート制限。

エラーが返った場合は エラーハンドリング を参照してください。
